# bilibili 弹幕屏蔽规则清单
- `(草|操|🌿|艹)`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `我操`
    - `太生草了`
    - `🌿`
    - `艹`
- `(顿|炖|遁|盾|挪)(出|煮)红(尘|城)`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `顿出红尘`
    - `炖煮红尘`
    - `遁出红尘`
    - `盾出红尘`
    - `挪出红尘`
    - `炖出红城`
- `(没|难)?(蚌|崩|绷)(埠|不)?(住)?(了)?`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `蚌`
    - `蚌埠住了`
    - `崩不住了`
    - `没绷住`
    - `难蚌`
- `(爷|奶)(青|童)(结|回)`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `爷青结`
    - `奶青回`
    - `爷童回`
- `(一|亿)拍(即|既)合`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `一拍即合`
    - `亿拍即合`
    - `亿拍既合`
- `((八|巴)人|迈)(八|巴|拔)(鹤|鶴|赫|河|盒|核)`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `八人八鹤`
    - `八人巴赫`
    - `八人拔河`
    - `八人八鶴`
    - `八人八盒`
    - `八人八核`
    - `巴人八鹤`
    - `迈巴鹤经销商`
- `((唯|惟|但)手熟|自动挡)尔`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `唯手熟尔`
    - `惟手熟尔`
    - `无它，但手熟尔`
    - `无他自动挡尔`
- `(?<!乾)坤`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `一坤年`
    - `一百坤`
- `.+播放.+赞`
  - 类型：正则
  - 是否启用：是
  - 示例：
    - `一播放13赞`
- `.+の.+`
  - 类型：正则
  - 是否启用：是

- `[Ss][Hh][Ii][Tt]`
